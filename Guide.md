# Транскриптор для Google Meet

## Шаг 1 — Установка расширения

1. Распаковать содержимое архива `extension` в любую папку
2. Открыть браузер и перейти на вкладку **Расширения → Управление расширениями**
3. В правом верхнем углу включить **Режим разработчика**
4. Нажать **Загрузить распакованное** и выбрать папку, куда мы экспортировали `extension`

## Шаг 2 — Запуск приложения

В зависимости от вашей системы открываем:
- `Meet Transcript win` — для Windows
-  Распаковать содержимое архива `Meet Transcript mac` и запустить новый файл `Meet Transcript mac` — для Mac



> **Важно для Mac:** если появляется сообщение _«Приложение повреждено»_ — нажмите правой кнопкой мыши на приложение → **Открыть** → **Открыть**. Это нужно сделать только один раз.

> **Важно для Mac 2:** если появляется окно _«Apple не удалось подтвердить, что файл «Meet Transcript mac» не содержит вредоносного ПО»_ — перейдите в 
**Системные настройки → Конфиденциальность и безопасность**. В нижней части окна будет строка: _«Файл "Meet Transcript mac" заблокирован для защиты вашего Mac»_ — нажмите **Открыть**.

> **Важно для Windows:** при первом запуске появится окно _«Windows защитила ваш компьютер»_ (SmartScreen). Нажмите **Подробнее** → **Всё равно выполнить**. Это стандартное предупреждение для приложений без платной подписи — приложение безопасно.

## Шаг 3 — Установка ffmpeg (только Windows, один раз)

Приложение использует `ffmpeg` для обработки аудио. На Windows его нужно установить один раз.

### Способ 1 — Через `ffmpeg.bat`
Просто запустите от имени администратора должно сработать

### Способ 2 — через winget (рекомендуется, встроен в Windows 10/11)

1. Нажмите **Win + R**, введите `cmd`, нажмите Enter
2. В открывшемся окне введите:
   ```
   winget install ffmpeg
   ```
3. Дождитесь завершения установки и перезапустите приложение

### Способ 3 — через Chocolatey

Если установлен [Chocolatey](https://chocolatey.org):
```
choco install ffmpeg
```

### Способ 4 — вручную

1. Перейдите на [github.com/BtbN/FFmpeg-Builds/releases](https://github.com/BtbN/FFmpeg-Builds/releases)
2. Скачайте файл `ffmpeg-master-latest-win64-gpl.zip`
3. Распакуйте архив
4. Из папки `bin` скопируйте `ffmpeg.exe` и `ffprobe.exe` в папку `C:\ffmpeg\bin\`
5. Добавьте `C:\ffmpeg\bin` в системную переменную PATH:
   - **Win + R** → `sysdm.cpl` → вкладка **Дополнительно** → **Переменные среды**
   - В блоке **Системные переменные** найдите `Path` → **Изменить** → **Создать** → вставьте `C:\ffmpeg\bin`
   - Нажмите **ОК** везде и перезапустите приложение

## Шаг 4 — Начало записи

Входим на Google Meet и в окне расширения слева в углу нажимаем **Start**
