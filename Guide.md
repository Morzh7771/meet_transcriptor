# Транскриптор для Google Meet

## Шаг 1 — Установка расширения

1. Распаковать содержимое архива `extension` в любую папку
2. Открыть браузер и перейти на вкладку **Расширения → Управление расширениями**
3. В правом верхнем углу включить **Режим разработчика**
4. Нажать **Загрузить распакованное** и выбрать папку, куда мы экспортировали `extension`

## Шаг 2 — Запуск приложения

В зависимости от вашей системы открываем:
- `Meet Transcript win` — для Windows
-  Распаковать содержимое архива `Meet Transcript mac` и запустить новый файл `Meet Transcript mac` — для Mac



> **Важно для Mac:** если появляется сообщение _«Приложение повреждено»_ — нажмите правой кнопкой мыши на приложение → **Открыть** → **Открыть**. Это нужно сделать только один раз.

> **Важно для Mac 2:** если появляется окно _«Apple не удалось подтвердить, что файл «Meet Transcript mac» не содержит вредоносного ПО»_ — перейдите в 
**Системные настройки → Конфиденциальность и безопасность**. В нижней части окна будет строка: _«Файл "Meet Transcript mac" заблокирован для защиты вашего Mac»_ — нажмите **Открыть**.

## Шаг 3 — Начало записи

Входим на Google Meet и в окне расширения слева в углу нажимаем **Start**
